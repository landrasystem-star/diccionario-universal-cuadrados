# 📦 Diccionario Universal de Cuadrados (DUC)
### Un lenguaje visual para que humanos y máquinas se entiendan sin palabras

---

## 🔍 ¿Qué es esto?

El **Diccionario Universal de Cuadrados (DUC)** es un lenguaje visual basado en patrones simples de cuadrados (`█` y espacios) que **cualquier humano puede entender y cualquier máquina puede reconocer**.

Cada cuadrado representa una **situación, emoción o necesidad universal**.

---

## 🌍 ¿Por qué existe?

Los humanos tenemos miles de idiomas que nos dividen. Las máquinas pueden tener **un solo lenguaje visual** que nos una.

Este diccionario es:
- ✅ **Libre** (MIT License)
- ✅ **Abierto** (cualquiera puede usarlo)
- ✅ **Gratuito** (para siempre)
- ✅ **Universal** (sin traducción, sin internet)

---

## 🧠 ¿Cómo funciona?

1. Una persona muestra un cuadrado (dibujado, impreso, en una pantalla)
2. Una máquina con cámara lo ve
3. La máquina busca el cuadrado en el diccionario
4. Ejecuta la acción correspondiente (encender luz, abrir puerta, pedir ayuda...)

**Sin botones. Sin comandos de voz. Sin aplicaciones.**

## 💡 Inspiración histórica

Este sistema continúa la evolución de los lenguajes universales:

| Año | Invento | Principio |
|-----|---------|-----------|
| 1830 | Código Morse | Puntos y rayas para comunicar letras |
| 1870 | Código Baudot | 5 bits para teletipos |
| 1963 | ASCII | 7 bits para ordenadores |
| 1994 | Código QR | Matriz 2D para datos |
| **2025** | **DUC** | **Patrones visuales para situaciones humanas** |

## 🔧 ¿Cómo usarlo?

### Para humanos:
- Memoriza los cuadrados principales (5 minutos)
- Dibújalos en un papel o imprímelos en tarjetas
- Muéstralos a una cámara

### Para máquinas (ESP8266):
```cpp
if (cuadrado == "█  █\n████\n█  █\n█  █") {
  digitalWrite(PIN_LUZ, HIGH);  // Enciende luz
}
